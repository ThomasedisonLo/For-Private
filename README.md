# 小游戏：迷宫冒险

## 介绍
欢迎来到迷宫冒险！你将穿越神秘的迷宫，寻找隐藏在其中的宝藏。但要小心，迷宫充满了未知的危险。你准备好了吗？

## 规则
- 使用以下指令控制你的冒险：
  - `北`：向北移动
  - `南`：向南移动
  - `东`：向东移动
  - `西`：向西移动
- 你会遇到三种情况：
  1. 🚪 门：选择一个方向进入下一个房间。
  2. ❓ 未知：探索未知的区域，可能有惊喜，也可能有危险。
  3. 💰 宝藏：你找到了宝藏，获胜！
